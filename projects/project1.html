Portfolio-design

Beskrivning

Det här projektet fokuserade på att skapa en modern, mörk portfolio med tydlig typografi och responsiv layout. Jag använde CSS-variabler, Flexbox och transitions för att skapa ett enhetligt intryck.

Utmaning

En utmaning var att få bilder att bibehålla proportioner på små skärmar. Lösning: object-fit: cover; + media queries för att ändra layout till kolumn.
