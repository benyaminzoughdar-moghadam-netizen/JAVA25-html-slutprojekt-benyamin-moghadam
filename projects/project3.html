Fotogalleri med Flexbox

Beskrivning

Detta projekt visar ett fotogalleri byggt med Flexbox som skalar snyggt över skärmstorlekar. Bilderna använder object-fit för att behålla proportioner utan att förvrängas.

Utmaning

Att undvika högupplösta bilder på mobil var viktigast. Lösning: ladda mindre bilder via media queries eller använd ng-ladda mindre upplösning för mobiler.
